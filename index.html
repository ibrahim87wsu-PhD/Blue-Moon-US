How to Host Your Website on Google Cloud

Step 1: Set Up Google Cloud Account

Create a Google Cloud account and activate the free trial ($300 credit for 90 days). Enable billing and create a new project for your website.

Step 2: Purchase & Configure Domain

Since you already own bluemoon.us.com, go to Google Domains or use Cloud DNS to configure your domain. Update DNS records to point to Google Cloud.

Step 3: Choose Hosting Option

For an e-commerce site, consider Google Cloud Storage (static site) or Compute Engine (dynamic). For this HTML site, Cloud Storage is easiest and cheapest.

Step 4: Upload Your Website

Create a Cloud Storage bucket named "www.bluemoon.us.com", set it for web hosting, and upload this HTML file along with CSS, JS, and image files.

Step 5: Configure SSL & Load Balancer

Set up Cloud CDN and Load Balancer with SSL certificate for secure HTTPS access. This ensures fast, secure delivery of your website globally.

Step 6: Launch & Monitor

Test your site at the temporary URL, then update DNS to point to your Cloud Storage endpoint. Set up monitoring with Cloud Monitoring.
